About

This application creates random tests for Front End Web Technical Interviews.

The original source for the questions is from the h5bp Front End Developer Interview Questions project.

I'm currently doing a manual grab of the source README.mdn, running it through a Markdown Processor (http://dillinger.io/), and then parsing the results into a list of JS objects. It is not ideal but the best short-term solution I could come up with. If you have any comments, please open an issue. Thanks!
